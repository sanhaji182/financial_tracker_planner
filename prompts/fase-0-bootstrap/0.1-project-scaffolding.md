## Prompt

```
Baca file berikut sebagai konteks sebelum mulai:
- AGENTS.md (rules & conventions)
- context/architecture.md (arsitektur sistem)

Buat project monorepo untuk Financial Operating System dengan struktur:
- /frontend — React + TypeScript + Vite, Tailwind CSS, light mode default + dark mode toggle
- /backend — Golang REST API dengan struktur clean architecture (handler, service, repository, model, middleware)
- /worker — Python service untuk OCR, PDF parsing, dan forecast analysis
- /docker — Docker Compose untuk PostgreSQL, Redis, dan semua service
- /docs — Tempat dokumentasi

Setup Detail:

1. **Frontend** (React + TypeScript + Vite):
   - Inisialisasi dengan Vite: `npx -y create-vite@latest ./ --template react-ts`
   - Install dependencies: Tailwind CSS v3, React Router v6, Zustand, @tanstack/react-query, Recharts, axios
   - Setup Google Font Inter
   - Buat folder structure: components/(ui, layout, shared), pages, hooks, services, stores, types, utils, constants, assets
   - Buat layout dasar: AppShell dengan Sidebar (260px) + TopBar (56px) + Main Content
   - Buat komponen dasar: Button (primary/secondary/ghost/danger), Input, Card, Badge, Modal, Table skeleton
   - Setup dark mode toggle menggunakan data-theme attribute
   - Setup design tokens sebagai CSS variables sesuai design system
   - Setup axios instance dengan base URL dan auth interceptor placeholder

2. **Backend** (Golang + Gin):
   - `go mod init github.com/user/financial-os`
   - Install: gin, pgx (PostgreSQL driver), golang-migrate, jwt-go, bcrypt, zerolog, godotenv, validator
   - Buat folder structure: cmd/server/main.go, internal/(handler, service, repository, model, dto, middleware, config, util)
   - Setup main.go dengan Gin router, middleware (CORS, logging, recovery)
   - Setup database connection pool (pgx)
   - Setup Redis connection
   - Setup config loader dari environment variables
   - Buat health check endpoint: GET /api/v1/health
   - Buat router group: /api/v1/

3. **Worker** (Python + FastAPI):
   - Buat main.py dengan FastAPI app
   - Buat folder structure: app/(api, services, models, utils)
   - requirements.txt: fastapi, uvicorn, pydantic, python-multipart, Pillow, pytesseract, pdfplumber, httpx
   - Buat health check endpoint: GET /health
   - Setup CORS middleware

4. **Docker**:
   - docker-compose.yml dengan services:
     - postgres:16 (port 5432, volume mount, init script)
     - redis:7-alpine (port 6379)
     - backend (build from Dockerfile.backend, port 8080)
     - frontend (build from Dockerfile.frontend, port 5173 dev)
     - worker (build from Dockerfile.worker, port 8081)
   - Dockerfile.backend (multi-stage: build + runtime)
   - Dockerfile.frontend (node:20-alpine)
   - Dockerfile.worker (python:3.11-slim + tesseract-ocr)
   - docker-compose.override.yml untuk development (hot reload)

5. **Config**:
   - .env.example dengan SEMUA environment variables (lihat architecture.md)
   - .gitignore (node_modules, vendor, __pycache__, .env, uploads/, *.log)
   - README.md dengan instruksi setup lengkap

6. **Verifikasi**:
   - `docker-compose up` harus berhasil menjalankan semua services
   - Frontend accessible di http://localhost:5173
   - Backend health check di http://localhost:8080/api/v1/health
   - Worker health check di http://localhost:8081/health
   - PostgreSQL dan Redis connected

Jangan buat fitur apapun — hanya scaffolding dan placeholder. 
Pastikan setiap service bisa dijalankan secara independen.
```
