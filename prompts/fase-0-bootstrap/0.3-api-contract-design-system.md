BAGIAN 2: Design System Implementation
═══════════════════════════════════════════

Implementasikan design system di frontend berdasarkan ui-design-system.md:

1. **CSS Variables** (/frontend/src/index.css):
   - Semua color tokens (light + dark mode)
   - Typography scale
   - Spacing scale
   - Shadow variables
   - Border radius variables
   - Transition variables

2. **Base Components** (/frontend/src/components/ui/):
   Setiap komponen harus:
   - TypeScript dengan proper Props interface
   - Mendukung className prop untuk customization
   - Memiliki variants sesuai design system
   - Berfungsi di light dan dark mode

   Komponen yang harus dibuat:
   - Button.tsx — variants: primary, secondary, ghost, danger, success; sizes: sm, md, lg; loading state
   - Input.tsx — variants: default, error; with label, helper text, error message
   - Card.tsx — with optional header, footer, padding variants
   - Badge.tsx — variants: success, warning, danger, info, transfer, ai; pill shape
   - Modal.tsx — sizes: sm, md, lg, xl; with overlay, close button, animations
   - Drawer.tsx — side panel, slide from right, 420px width
   - Table.tsx — with header, sortable columns, skeleton loading rows
   - Select.tsx — dropdown select with search
   - Checkbox.tsx
   - Toggle.tsx — for switches/toggles
   - Alert.tsx — variants: info, warning, danger, success; with action button
   - Toast.tsx — notification system (top-right, auto-dismiss)
   - Skeleton.tsx — loading skeleton shapes (text, card, table row)
   - EmptyState.tsx — illustration + message + CTA button
   - ErrorState.tsx — error icon + message + retry button
   - Tooltip.tsx
   - Dropdown.tsx — dropdown menu
   - Tabs.tsx
   - Pagination.tsx

3. **Layout Components** (/frontend/src/components/layout/):
   - AppShell.tsx — main layout wrapper
   - Sidebar.tsx — 260px navigation sidebar (collapsible on mobile)
   - TopBar.tsx — 56px height, logo, search, bell icon, user avatar
   - PageHeader.tsx — page title + breadcrumb + action buttons
   - ContentGrid.tsx — responsive grid wrapper

4. **Shared Components** (/frontend/src/components/shared/):
   - MoneyDisplay.tsx — format "Rp 5.200.000" dengan warna dan konteks
   - PercentageChange.tsx — "↑ 8%" hijau atau "↓ 3%" merah
   - StatusBadge.tsx — paid/unpaid/overdue/active/inactive badges
   - DateDisplay.tsx — format tanggal Indonesia
   - AvatarInitials.tsx — avatar dari inisial nama
   - SearchBar.tsx — search input dengan debounce
   - FilterBar.tsx — composable filter chips

5. **Hooks** (/frontend/src/hooks/):
   - useTheme.ts — dark/light mode toggle
   - useToast.ts — toast notification hook
   - useModal.ts — modal open/close state
   - useDebounce.ts — debounce for search
   - usePagination.ts — pagination state
   - useAuth.ts — auth context hook (placeholder)

6. **Services** (/frontend/src/services/):
   - api.ts — axios instance with base URL, auth interceptor, error interceptor
   - auth.ts — login, register, refresh, logout functions (placeholder)

7. **Stores** (/frontend/src/stores/):
   - authStore.ts — Zustand store for auth state (user, tokens, isAuthenticated)
   - themeStore.ts — Zustand store for theme preference
   - toastStore.ts — Zustand store for toast notifications

Buat juga satu halaman demo di /frontend/src/pages/DesignSystemDemo.tsx yang menampilkan semua komponen dalam satu halaman (untuk testing visual).
