## Prompt

```
Baca context: AGENTS.md, database-schema.md (forecasts), business-rules.md (bagian Forecast Cashflow dan Safe-to-Spend — ikuti formula PERSIS).

Implementasikan modul Forecast Cashflow:

═══ BACKEND ═══

Service (forecast_service.go):
- CalculateMonthlyForecast(ctx, userID, month) → ForecastResponse
- Kalkulasi persis sesuai business-rules.md:
  1. estimated_income: rata-rata income 3 bulan terakhir (atau income recurring)
  2. estimated_fixed_expenses: sum(bills.amount) + sum(debts.minimum_payment) bulan target
  3. estimated_variable_expenses: rata-rata variable expenses 3-6 bulan per kategori
  4. daily_projections: array [{date, projected_balance}]
     - Start dari cash_available hari ini
     - Setiap hari: kurangi bills jatuh tempo + estimated daily variable
     - Tambahkan income pada tanggal expected (1 atau 25 dll)
  5. projected_end_balance: saldo hari terakhir bulan
  6. lowest_balance: minimum dari daily_projections
  7. lowest_balance_date: tanggal saldo terendah
  8. safe_to_spend: income - fixed - (variable × 80%) - (5% emergency reserve)
  9. is_tight: true jika projected balance < 1× monthly_living_cost pada tanggal manapun

- Cache forecast di Redis (TTL: 30 menit)
- Simpan juga ke DB (forecasts table) untuk historical comparison

Handler:
- GET /api/v1/forecast/monthly?month=2026-07 → full forecast
- GET /api/v1/forecast/daily?month=2026-07 → daily projections array

═══ FRONTEND ═══

Page: ForecastPage.tsx (/forecast)

Top summary cards:
| Est. Income | Fixed Expenses | Variable Est. | Projected Balance | Safe to Spend |
| Rp 25jt     | Rp 15jt        | Rp 5jt        | Rp 8.5jt          | Rp 3.2jt ✅   |

Main chart: Area/Line chart (Recharts)
- X: tanggal (1-30/31)
- Y: projected balance
- Area fill: gradient (hijau jika healthy, merah jika below threshold)
- Highlight titik: lowest_balance_date (dot merah + label)
- Horizontal line: threshold (1× monthly living cost)
- Tooltip on hover: tanggal + projected balance + event (jika ada bill/payment)

Warning banner (jika is_tight):
"⚠️ Proyeksi saldo turun di bawah Rp 5.000.000 pada tanggal 15 Juli 2026. Pertimbangkan menunda pengeluaran non-esensial."

Detail breakdown table:
| Tanggal | Event | Jumlah | Saldo Proyeksi |
| 1 Jul   | Gaji  | +25jt  | 35jt          |
| 5 Jul   | KPR   | -8jt   | 27jt          |
| 10 Jul  | Listrik| -800rb | 26.2jt       |
...

Month picker: navigate antar bulan

PENTING:
- Chart harus smooth dan informatif
- Lowest balance point harus SANGAT jelas (visual prominent)
- Safe-to-spend harus di dashboard juga
```
