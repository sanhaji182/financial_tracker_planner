## Prompt

```
Baca context: AGENTS.md, database-schema.md (household_notes, task_checklists), api-conventions.md.

Implementasikan empat fitur pendukung:

═══ 1. EXPORT ═══

Backend:
- GET /api/v1/export/transactions?format=csv&date_from=X&date_to=Y&account_id=Z
  - Generate CSV: Date, Type, Category, Account, Amount, Description
  - Stream download
- GET /api/v1/export/monthly-report?month=2026-07&format=pdf
  - Generate PDF dari monthly_closing data
  - Use Go PDF library (gofpdf atau similar)

Frontend:
- Export button di TransactionsPage → download CSV
- Export button di MonthlyClosingPage → download PDF

═══ 2. BACKUP & RESTORE ═══

Backend:
- POST /api/v1/backup/create → pg_dump → encrypt → save to storage
  - Return: backup metadata (date, size, filename)
- GET /api/v1/backup/list → list available backups
- POST /api/v1/backup/restore → upload encrypted backup → pg_restore
  - DANGEROUS: require confirmation + re-auth

Frontend: BackupPage.tsx (/settings/backup)
- List backups: date, size, download button
- "Buat Backup Sekarang" button
- "Restore" button with double confirmation ("Ini akan mengganti SEMUA data. Lanjutkan?")

═══ 3. HOUSEHOLD JOURNAL ═══

Backend:
- CRUD /api/v1/journal → household_notes
- Filter: date range, tags, search

Frontend: JournalPage.tsx (/journal)
- Timeline view: entries sorted by date desc
- Card per entry: title, date, content preview, tags
- Create/edit: title, date, rich text content (markdown), tags
- Search + filter by tag
- Example entries: "Juli 2026: Memutuskan refinance KPR", "Mulai investasi reksadana di Bibit"

═══ 4. TASK CHECKLIST ═══

Backend:
- CRUD /api/v1/tasks → task_checklists
- Auto overdue: cron marks tasks WHERE due_date < today AND status='pending' → 'overdue'
- Filter: status, due_date range, frequency

Frontend: TasksPage.tsx (/tasks)
- Checklist view: checkbox + title + due date + frequency badge
- Sorting: due date ascending (soonest first)
- Overdue items: highlighted red
- Completed: strikethrough + green check
- Quick add form
- Examples: "Bayar PBB", "Perpanjang asuransi mobil", "Review portofolio Q3"
```
