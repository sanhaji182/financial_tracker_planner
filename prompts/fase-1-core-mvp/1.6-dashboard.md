## Prompt

```
Baca SEMUA context files — ini adalah prompt paling penting karena dashboard adalah jantung aplikasi.

Implementasikan Dashboard Utama yang menjawab 3 pertanyaan: "Berapa kondisi keuangan saya?", "Apa yang terjadi bulan depan?", "Uang sisa diapakan?"

═══ BACKEND ═══

Service (dashboard_service.go):
- GetDashboardData(ctx, userID) → DashboardResponse
- Kalkulasi SEMUA metrik sesuai business-rules.md:
  1. net_worth = total aset - total utang
  2. total_assets (breakdown: liquid, invested, property)
  3. total_debts (count + total outstanding)
  4. cash_available = sum saldo akun liquid (bank, e_wallet, cash)
  5. dti_ratio = total min debt payments / total income bulan ini × 100
  6. health_score = formula komposit (DTI 30% + EF 30% + Cash 20% + Savings 20%)
  7. upcoming_bills_7_days = query bills WHERE next_due_date BETWEEN today AND today+7
  8. forecast_end_of_month = estimasi saldo akhir bulan (simplified)
  9. safe_to_spend = income - committed expenses - bills - buffer
  10. recent_alerts = top 5 unread alerts
  11. monthly_insight_summary = "Pengeluaran makan naik 15% bulan ini"
  12. next_action = prioritas tertinggi dari allocation advice rules

- Cache result di Redis (key: dashboard:{userID}, TTL: 5 menit)
- Invalidate cache saat ada perubahan transaksi/akun/utang

Handler:
- GET /api/v1/dashboard → return DashboardResponse (cached)

DTOs (DashboardResponse):
```go
type DashboardResponse struct {
    NetWorth          MoneyValue     `json:"net_worth"`
    TotalAssets       AssetBreakdown `json:"total_assets"`
    TotalDebts        DebtSummary    `json:"total_debts"`
    CashAvailable     MoneyValue     `json:"cash_available"`
    DTIRatio          float64        `json:"dti_ratio"`
    DTIStatus         string         `json:"dti_status"` // healthy, warning, danger
    HealthScore       HealthScore    `json:"health_score"`
    UpcomingBills     []UpcomingBill `json:"upcoming_bills"`
    ForecastEndMonth  MoneyValue     `json:"forecast_end_month"`
    SafeToSpend       MoneyValue     `json:"safe_to_spend"`
    RecentAlerts      []Alert        `json:"recent_alerts"`
    InsightSummary    string         `json:"insight_summary"`
    NextAction        NextAction     `json:"next_action"`
    NetWorthTrend     []TrendPoint   `json:"net_worth_trend"` // 6 bulan
}
```
