## Prompt

```
Baca context: AGENTS.md, database-schema.md (debts, debt_payments), api-conventions.md (Debts), business-rules.md (bagian Debt Avalanche Simulation).

Implementasikan modul Utang & Cicilan:

═══ BACKEND ═══

Service (debt_service.go):
- CreateDebt → validasi fields, set status 'active'
- RecordPayment:
  - Buat debt_payment record
  - Buat transaksi expense otomatis (linked ke debt_payment)
  - Update outstanding_balance: outstanding -= payment_amount
  - Jika outstanding <= 0 → set status 'paid_off'
  - Kalkulasi principal_portion dan interest_portion
  - ATOMIC semua operasi
- GetDebts → list filtered, sorted by interest_rate desc
- GetDebtSummary → total utang, total minimum payment, count active
- SimulateAvalanche(debts, extraMonthly):
  - Implementasi persis sesuai business-rules.md
  - Return: months_to_payoff, total_interest_paid, comparison_without_extra
  - Return per-debt payoff schedule

Handler:
- GET    /api/v1/debts → list
- GET    /api/v1/debts/:id → detail + payments history
- POST   /api/v1/debts → create
- PUT    /api/v1/debts/:id → update
- DELETE /api/v1/debts/:id → soft delete
- POST   /api/v1/debts/:id/payments → record payment
- GET    /api/v1/debts/summary → summary
- GET    /api/v1/debts/avalanche?extra=500000 → simulasi avalanche

═══ FRONTEND ═══

Page: DebtsPage.tsx (/debts)
- Top summary: Total Utang | Min Payment Bulan Ini | Jumlah Utang Aktif
- Cards per utang:
  - Nama, kreditur, tipe badge (KPR/CC/Cicilan)
  - Outstanding balance (MoneyDisplay)
  - Progress bar: (original - outstanding) / original × 100%
  - Due date (tanggal jatuh tempo bulan ini)
  - Interest rate: "12% p.a."
  - Min payment: "Rp 2.500.000/bln"
  - Warning badge jika due date < 7 hari
- Tombol "+ Tambah Utang"
- Modal form: nama, tipe, kreditur, total pinjaman, outstanding, interest rate, min payment, due_day, tenor, akun pembayaran, catatan

Detail view: DebtDetailPage.tsx (/debts/:id)
- Info lengkap utang
- Progress chart: pelunasan over time (bar chart)
- Payment history table: tanggal, jumlah, pokok/bunga, remaining balance
- Tombol "Bayar" → form: jumlah bayar, is_extra_payment toggle, catatan
  - Default jumlah = minimum_payment
  - Jika extra → label "Pembayaran Extra"

Avalanche Simulator: DebtAvalanchePage.tsx (/debts/avalanche)
- Input: jumlah extra payment per bulan (slider + input number)
- Tabel hasil:
  | Utang | Bunga | Outstanding | Tanpa Extra | Dengan Extra | Hemat |
  - Sorted by interest rate desc
- Summary: "Dengan extra Rp 500rb/bulan, semua utang lunas 18 bulan lebih cepat dan hemat bunga Rp 12.5jt"
- Chart: timeline pelunasan per utang (stacked bar)
```
